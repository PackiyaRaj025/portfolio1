To-Do List Web Application

This project is a To-Do List web app to manage daily tasks. Users can add new tasks, mark tasks as completed, and delete tasks. Tasks are saved in browser localStorage, so data is not lost after page refresh. This project helped me learn DOM manipulation, events, and localStorage usage in JavaScript.

Tech Stack Used:
  • HTML - page structure and input elements
  • CSS - styling and layout
  • Bootstrap - responsive design
  • JavaScript - add, delete, complete tasks
  • LocalStorage - save tasks in browser
Live Demo

Wikipedia Search Application

This project is a Wikipedia search web app. Users can type a keyword and press Enter to search. The app fetches data from Wikipedia API and shows title, link, and description. A loading spinner is displayed while data is loading. This project helped me learn API calling, fetch method, and dynamic DOM updates.

Tech Stack Used:
  • HTML - page structure and input elements
  • CSS - styling and layout
  • Bootstrap - responsive design
  • JavaScript - API call and DOM manipulation
  • Fetch API - get data from Wikipedia
